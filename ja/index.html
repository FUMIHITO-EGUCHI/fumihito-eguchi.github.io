OTYATYA

ソフトウェア開発者 / 自動化ツール制作

概要

個人開発者。Claude Code 用のスキル、Chrome 拡張、業務自動化スクリプト、 ゲーム MOD などを作っている。基本的には自分が普段使いしていて 「毎回これ面倒だな」と思った部分を、自分のために潰すための小さいツールが多い。

技術はタスクに合わせて選ぶ。最近触っているのは Python / TypeScript / シェル / C# / Lua / HTML+CSS あたり。 コードは Claude Code と Codex を併用して書いている。

受託開発もときどき引き受けています。async-first、JST 基準。下記の Email まで気軽にどうぞ。

プロジェクト

Claude Skills 群

Claude Code / Claude デスクトップ向けの自作スキル群。メルカリの相場調査と 出品フォーム自動入力など、日常の手作業を自動化するスキルを継続開発中。

Claude Skills Python Cowork

x-post-archive-extension

X（旧Twitter）の投稿をローカルに保存するChrome拡張。 自分のいいねやブックマークが後から探しづらい問題を、 手元で気軽に検索できる形にして補うのが狙い。 現在 Chrome ウェブストア（CWS）登録に向けて準備中。

Chrome Extension TypeScript GitHub →

Workshop MOD 日本語化パック [B41]

Project Zomboid Build 41 向けの、複数 Workshop MOD をまとめて日本語化する MOD。 翻訳テキストに加え、対象 MOD ごとの Lua パッチでメニュー名やツールチップも 日本語化する。現在開発中（非公開リポ）。

Game MOD Lua Project Zomboid

EraserMod (STS2_oekaki_patch)

Slay the Spire 2 のマップ落書き機能を拡張する MOD。消しゴム幅・鉛筆の太さ・色・ Undo を調整できるようにする。Harmony で実行時パッチを当て、公式 MOD ローダー 経由でロード。マルチプレイにも対応。現在開発中。
