About概要

個人開発者。Claude Code 用のスキル、Chrome 拡張、業務自動化スクリプト、 ゲーム MOD などを作っている。基本的には自分が普段使いしていて 「毎回これ面倒だな」と思った部分を、自分のために潰すための小さいツールが多い。

技術はタスクに合わせて選ぶ。最近触っているのは Python / TypeScript / シェル / C# / Lua / HTML+CSS あたり。 コードは Claude Code と Codex を併用して書いている。

Solo developer based in Japan. I build small things — Claude skills, Chrome extensions, automation scripts, and game mods — mostly to scratch my own itches. Stack varies per task; recently Python, TypeScript, shell, C#, Lua, HTML+CSS. Coding is done with Claude Code and Codex side by side.

EraserMod (STS2_oekaki_patch) Slay the Spire 2 MOD

Slay the Spire 2 のマップ落書き機能を拡張する MOD。消しゴム幅・鉛筆の太さ・色・ Undo を調整できるようにする。Harmony で実行時パッチを当て、公式 MOD ローダー 経由でロード。マルチプレイにも対応。現在開発中（非公開リポ）。

A mod for Slay the Spire 2 that improves the in-game map drawing tools — adjustable eraser width, pencil thickness and color, and undo. Loaded through the official mod manager with Harmony runtime patching, multiplayer-aware. In development, private repository.
